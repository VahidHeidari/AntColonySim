This is a very simple ant colony simulator :)

 Keys
------

* A -> Pause/Unpause simulation
* D -> Toggle drawing sensing area
* Escape -> Quit
* N -> Toggle drawing ant numbers
* S -> Simulation speed
* T -> Toggle drawing trails
